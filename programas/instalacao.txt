Passos para configuração da máquina:

1 - Instalar Git:
-Executar o arquivo 'Git-2.20.1-32-bit.exe'

2 - Clonar projeto do Git: 'https://github.com/DanielTorres31/miaudote.git'

3 - Instalação Node.js
-Executar arquivo 'node-v8.0.00x64.msi'
-Após a conclusão, execute no cmd para conferir: $ node -v

4 - Instalação do angular-cli
-Execute no cmd: $ npm install -g @angular/cli
-Após a conclusão, execute no cmd para conferir: $ ng version

5 - Instalação Visual Studio Code (Opcional)
-Executar arquivo 'VSCodeUserSetup-ia32-1.30.1.exe'

6 - Instalar dependências do projeto:
-Abra o cmd no **diretório do projeto** e execute: $ npm install
